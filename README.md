# 啟動

React 開發。

請先安裝 Node 與 Npm。

並輸入下面的指令安裝 modules。

```
$ npm install
```

啟動開發伺服器

```
$ npm start
```

或是打包過後的檔案啟動伺服器

```
$ npm run build
$ serve -s build
```
